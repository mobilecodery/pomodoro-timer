# ⏳ Focus Flow: My Pomodoro Timer
> "Building tools to manage time, while I write my first novel."

## 🚀 About the Project
I am a student from Pakistan (Sindh Board) preparing for college in 2026. I built this Pomodoro Timer to solve my own problem: staying focused while balancing coding, a crochet business, and writing.

## 🛠️ Tech Stack
- ***Structure:*** HTML5
- ***Style:*** CSS3 (Working on a minimalist aesthetic)
- ***Logic:*** JavaScript (Vanilla)
